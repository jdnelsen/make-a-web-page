Lesson 1 - Basics of the Web and HTML

What is a Web Page

A web page is a text document written in a language called HTML. Web browsers read these documents, and then interpret and display them.

How Coding Works

Coding happens when programmers write text in a language that a computer can understand. The computer can then follow the instructions the programmer wrote. Programmers need to write exactly the way a computer understands

HTML

Hypertext Markup Language. HTML is the main type of document on the web.

Tag

An HTML tag is always contained within angled brackets. Most tags have an opening tag and a closing tag, Some tags (called "void" tags) do not require a closing tag

Element

An HTML element refers to everything within a set of opening and closing tags. Some elements are for inline content while others for block. Inline simply means that the content is viewed as being on the same line or lines, while block content has padding around it placing it in its own invisible content box. Every element has a box.

Lesson 2 - Webpage, Documents, and Structures

Developer Tools

Every element has a box. Span are inline and DIV are block. Block elements form an invisible box around them.

CSS

Allows us to use syntax and rules to style the page

DOM

Is a tree like structure that has a way of representing the hierarchical nature of a structure in a graphical form. Boxes inside of boxes.

Lesson 3 - Adding CSS Style to HTML Structure

What is CSS

CSS is code written to control the style of HTML elements

Boxes

HTML elements are boxes and each box has 4 components. Different browsers work slightly differently. Sometimes this causes different browsers to display the same code differently. Divs are block elements, o by default they take up the entire width of a page.

Code, Test, Refine

First looks for natural boxes. Look for repeated Styles and elements. Write the HTML. Apply the styles largest to smallest, then fix the small things. Do this until you are happy. Developer tools are a great way to do this. Use an HTML and CSS verifier.
